MdxEscaper

object MdxEscaper

MDX "escaping-hell" is really FOUR channels with different rules. Routing each correctly is what retires the problem instead of hoping it away.

Types

HtmlFallback
Link copied to clipboard
object HtmlFallback

Channel 3 — the raw-HTML fallback branch, the real sharp edge. Raw HTML in MDX is JSX: void tags must self-close, attributes must be quoted, and bare <, &, { in text break compilation. This renders a subtree as JSX-VALID markup so the fallback can never crash the build.
